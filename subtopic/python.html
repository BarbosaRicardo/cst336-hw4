mysql.connector
  • The front-end application send a connection request to MySQL's connector API

  • The API then forwards the same request to the database

  • The database accepts the request and send the connected message

  • Python application send cursor-connection request. This allows one to create their own database and helps with communication through the entire database MySQL server

  • The python application executes a certain MySQL statement or query and the database will fetch that data
