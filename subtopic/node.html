Getting Ready!


You will need to do two things before you can start with MySQL in Node.JS: make sure MySQL is installed and install npm module.

  • First, import the mysql module
  • Second, create a connection to the MySQL database that calls the createConnection() method; which has 4 parameters to fill out: host, user, password, and database.
  • Third, call the connect() method on the connection object to connect to the MySQL database server. The connect() method will accept a callback function that has the error argument what provides debugging detail.
  • Finally, to close the connection, one must call the end() method on the connection object. This ensures that the queries remaining are always executed before the database connection closed.
