Getting Ready!


To connect to MySQL in Java, MySQL implements the Java Database Connection (JDBC) API with a driver. The JDBC API defines how a client may apply CRUD operations; oriented toward relational databases.

  • First, attain a driver class for the mysql database
  • Second, identify the connection URL for the mysql database using the API, type of database, server name, port number, and the database name.
  • Third, locate username and password for the mysql database (i.e., MySQL workbench credentials)
  • Finally, to close the connection.
